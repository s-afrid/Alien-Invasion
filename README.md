Description:

Developed a classic arcade-style game in Python using the Pygame library.
Implemented a player-controlled spaceship that moves right and left based on user input.
Integrated a firing mechanism allowing the player to shoot bullets in response to input commands.
Designed a fleet of alien enemies with interactive behaviors for an engaging gaming experience.
Iteratively refined the game, introducing limits on the number of player ships allowed.
Added advanced features such as a dynamic scoreboard to track player performance.
Technologies Used:

Python
Pygame library
Key Features:

Player-controlled spaceship with right and left movement.
Firing mechanism for bullets in response to player input.
Dynamic fleet of interactive alien enemies.
Implementation of limits on the number of player ships.
Scoreboard to track and display player performance.
